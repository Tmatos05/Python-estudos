# Sistema-de-M-dia-Escolar-em-Python
Este projeto em Python calcula a média de duas notas informadas pelo usuário e determina a situação do aluno (Aprovado, Recuperação ou Reprovado) com base no valor da média.  O objetivo é praticar conceitos básicos da linguagem Python, como entrada de dados, operações matemáticas e estruturas condicionais (if, elif, else).
